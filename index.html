Summary:

Event propagation determines the order nested elements receive events they all have a handle on

  • Bubbling - event is first captured and handled by the innermost element, then 'bubbles' up to the respective parent elements. This method is the default.

  • Capture - (opposite of bubbling) event is first captured by the outermost element, before it gets passed to the inner elements. To use this, set option 'capture' equal to 'true'.

  • Once - similar to unbinding itself. For example, if you have a button that should only be clicked once, can set option 'once' equal to 'true'.
